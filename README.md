# My Portfolio 📄

A **minimalistic** portfolio website with elegant serif typography, built with Next.js 14, TypeScript, and TailwindCSS.

## Features

- 🎨 **2-Color Design** - Deep blue (#1e3a8a) and dark gray (#374151) only
- 📊 **Scroll Progress Bar** - Subtle progress indicator at the top
- 🎭 **Smooth Parallax Scrolling** - Elegant depth-based motion effects
- ✨ **Subtle Animations** - Refined, non-distracting transitions
- 🖋️ **Serif Typography** - Classic Georgia font for elegance
- 🎯 **Link Underlines** - Clean animated underlines on hover
- 💼 Work Experience with minimal timeline
- 🚀 Projects showcase with clean design
- 📝 Blog section with clean typography
- 📱 Fully responsive design
- ⚡ Built with Next.js 14 App Router
- 🔤 TypeScript for type safety
- 💅 Styled with TailwindCSS
- 🎬 Framer Motion for smooth parallax

## Color Palette

- **Primary (Deep Blue)**: `#1e3a8a`
- **Secondary (Dark Gray)**: `#374151`, `#4b5563` (light)
- **Background**: White `#ffffff`

## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Adi16023/My-Portfolio.git
cd My-Portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Build for Production

```bash
npm run build
npm start
```

## Customization

- Update personal information in the components
- Modify color scheme in `tailwind.config.js`
- Add your projects, experience, and blogs in respective component files
- Replace placeholder links with your actual social media profiles

## Technologies Used

- **Next.js 14** - React framework with App Router
- **TypeScript** - Type safety
- **TailwindCSS** - Utility-first styling
- **Framer Motion** - Animation library
- **React Intersection Observer** - Scroll-triggered animations
